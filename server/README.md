# Telegram App Admin Bot

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. Создай бота через @BotFather в Telegram
2. Получи токен бота
3. Узнай свой Telegram ID через @userinfobot
4. Замени в `bot.py`:
   - `YOUR_BOT_TOKEN_HERE` на токен бота
   - `ADMIN_IDS = [123456789]` на свой ID

## Запуск

```bash
python bot.py
```

## Функции

- 📊 Статистика приложения
- 👥 Управление пользователями
- 💬 Просмотр сообщений (зашифрованных)
- 🔒 Настройки шифрования (Fernet AES-128)
- ⚙️ Настройки приложения

## Команды

- `/start` - Главное меню
- `/adduser <id> <name>` - Добавить пользователя
- `/send <message>` - Отправить зашифрованное сообщение

## Безопасность

- Доступ только для админов по Telegram ID
- End-to-end шифрование сообщений
- Ключ шифрования генерируется автоматически
